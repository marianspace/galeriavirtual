Proyecto Final del curso de Python para Coderhouse

Proyeco:
El proyecto es una especie de Mercado libre de Arte donde los artistas pueden publicar sus obras y los compradores adquirirlas en una plataforma más seria que las redes sociales
La base de usuario cuenta con dos grupos de personas, compradores y Artistas, estos últimos pueden postear, modificar sus obras
El proyecto está mejor que le vez anterior
